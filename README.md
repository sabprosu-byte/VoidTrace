# VoidTrace
Python tabanlı temel bir güvenlik analiz aracı
Red Hat / Linux Edition

## Özellikler
- Port Analizi (Port Tarama)
- Ağ Bilgisi Sorgulama
- Terminal Tabanlı Arayüz
- Target Intelligence (IP Konum ve Servis Sağlayıcı Analizi)
- HTTP Header Analizi
- Antivirüs (Dosya Analiz)

## Kurulum

Termux veya Linux üzerinde çalıştırmak için:

1. Depoyu klonlayın:
git clone https://github.com/sabprosu-byte/VoidTrace.git

2. Klasöre gidin:
cd VoidTrace

3. Aracı çalıştırın:
python voidtrace.py
